# RobustGCN
PyTorch implementation of "[Robust Graph Convolutional Networks Against Adversarial Attacks](https://zw-zhang.github.io/files/2019_KDD_RGCN.pdf)", KDD 2019. 

## Example Usage
```
python train.py
```
