# Graph Convolutional Networks
PyTorch implementation of [Semi-Supervised Classification with Graph Convolutional Networks](http://arxiv.org/abs/1609.02907) (ICLR 2017) by Thomas N. Kipf, Max Welling.

## Run the demo
```bash
python train.py
```
