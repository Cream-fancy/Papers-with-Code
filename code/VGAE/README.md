# Variational Graph Auto-Encoders
PyTorch implementation of variational graph auto-encoder by Thomas Kipf. For details of the model, refer to his original [tensorflow implementation](https://github.com/tkipf/gae) and [his paper](https://arxiv.org/abs/1611.07308).

## Run the demo
* Run `python train.py`
* You can specify your arguments in `args.py`
